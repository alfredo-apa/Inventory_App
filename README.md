# Inventory App

Inventory management project biult with Java (Core) and MySQL.
Designed as and internal enterprise tool for tracking assets, locations and assigments.

## Tech Stack
- Oracle OpenJDK 23.0.2 (Maven)
- MySQL

## Scope
- Relationl database schema and sample data scripts
- JDBC connection
- Basic CRUD operations

## How to Run
1. Create the database using the scripts in /sql
2. Configure DB credentials
3. Compile the project as an Artifact
4. Run the app Artifact
